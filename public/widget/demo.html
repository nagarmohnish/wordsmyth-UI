Go Ad-Free Widget — Live Demo

How it works: One script tag. The widget auto-detects every ad on the page and adds a witty, full-width bar above each one inviting users to go ad-free. Each ad gets a different message. Try hovering the green buttons!

Setup — One Line

<!-- Just add this before </body> -->
<script src="https://your-domain.com/widget/go-ad-free.js"
  data-subscribe-url="https://your-domain.com/subscribe"
></script>
