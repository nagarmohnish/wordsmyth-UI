Wordsmyth Widgets

Plug-and-play embeddable widgets for any website. Zero dependencies. Just one script tag.

1. Donation Widget

A universal support widget with preset amounts ($5, $10, $25, $50), custom amount input, one-time/monthly toggle, trust signals, and a CTA that links to your support page.

Sticky Placement Fixed bottom-left green pill that expands into the full donation card on click. Appears after a configurable delay.
Inline Placement Renders the donation card directly in page flow, wherever the script tag is placed. No floating elements.

Live Demo — Inline Widget

Widget renders below

Embed Code — Inline

<!-- Wordsmyth Donation Widget (inline) --> <script src="https://kids.wordsmyth.net/widget/donate.js" data-support-url="https://kids.wordsmyth.net/support" data-placement="inline"></script>

Embed Code — Sticky

<!-- Wordsmyth Donation Widget (sticky pill) --> <script src="https://kids.wordsmyth.net/widget/donate.js" data-support-url="https://kids.wordsmyth.net/support" data-placement="sticky" data-delay="3000"></script>

Configuration

AttributeDefaultDescription
data-support-url/supportWhere "Continue" button links to
data-placementstickysticky or inline
data-delay3000Delay (ms) before sticky pill appears
data-target—CSS selector for inline container element
data-utmdonation-widget-embedUTM source for analytics tracking

2. Go Ads-Free Widget

Automatically finds every ad on the page and overlays a witty "Go Ads-Free" bar. Detects Google Ads, Amazon, Mediavine, Freestar, and 40+ ad network patterns out of the box.

Auto-Detection Scans for ads by class, ID, data attributes, and iframe sources. No manual targeting needed.
Responsive Overlays Wide ads get witty text + green CTA pill. Narrow ads (skyscrapers) get a compact CTA-only bar.

Configuration

AttributeDefaultDescription
data-subscribe-url/subscribeWhere "Go Ads-Free" CTA links to
data-delay1500Delay (ms) before first ad scan
data-poll3000Re-scan interval for lazy-loaded ads (ms). Set 0 to disable.
data-selectors—Extra CSS selectors to target (comma-separated)
data-utm-paramutm_source=go-ad-free-widgetURL parameter appended to CTA link

3. Use Both Together

Drop both script tags on any page. The donation widget handles contributions, while the Go Ads-Free widget converts ad impressions into subscribers.

Combined Embed Code

<!-- Wordsmyth Donation Widget (sticky) --> <script src="https://kids.wordsmyth.net/widget/donate.js" data-support-url="https://kids.wordsmyth.net/support" data-placement="sticky"></script> <!-- Wordsmyth Go Ads-Free Widget --> <script src="https://kids.wordsmyth.net/widget/go-ad-free.js" data-subscribe-url="https://kids.wordsmyth.net/subscribe"></script>
Zero Dependencies Pure vanilla JS. No React, jQuery, or framework needed. Works on any website.
Lightweight ~8KB each unminified. Self-contained CSS injection. No external requests beyond the script.
Analytics Ready Fires Google Analytics events (gtag) for impressions, amount selections, clicks, and dismissals.
Fully Configurable All options via data-* attributes on the script tag. No code changes or build steps.
