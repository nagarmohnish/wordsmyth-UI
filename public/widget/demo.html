Sample Article Content

This demonstrates how the Go Ad-Free widget works on any website. Simply add one script tag and the widget automatically finds all ad placements on the page and adds a discreet "Go Ad-Free" button to each one.

The widget detects ads from all major networks: Google AdSense, Google Publisher Tags (GPT/DFP), Amazon Associates, Freestar, Mediavine, AdThrive, and any element using common ad-related CSS class names or data attributes.

When a user clicks the "Go Ad-Free" button, they are redirected to your subscription page. You configure the URL via a data-subscribe-url attribute on the script tag.

Setup Instructions

Add this single line before </body>:

<!-- Go Ad-Free Widget -->
<script
  src="https://your-domain.com/widget/go-ad-free.js"
  data-subscribe-url="https://your-domain.com/subscribe"
></script>

That's it. The widget handles everything else automatically.

Configuration Options

Attribute Default Description
data-subscribe-url /subscribe URL to redirect when button is clicked
data-label ✕ Go Ad-Free Button text
data-theme light light or dark
data-position top-right top-right, top-left, bottom-right, bottom-left
data-delay 1500 Milliseconds to wait before first scan (ads need time to load)
data-poll 3000 Re-scan interval in ms for late-loading ads. 0 to disable
data-selectors empty Extra CSS selectors to match (comma-separated)
data-utm-param utm_source=go-ad-free-widget UTM parameter appended to the redirect URL

What It Detects

The widget automatically detects ad elements from:

  • Google AdSense (ins.adsbygoogle)
  • Google Publisher Tags / DFP ([id^="div-gpt-ad"])
  • Amazon Associates ([id^="amzn-assoc"])
  • Freestar, Mediavine, AdThrive
  • Any element with ad-slot, ad-container, ad-banner, advertisement, or sponsored in class/id
  • Data attributes: data-ad, data-ad-slot, data-ad-unit
  • Ad iframes (DoubleClick, Google Syndication, Amazon)

Plus you can add custom selectors via data-selectors.
