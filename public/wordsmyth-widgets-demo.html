Wordsmyth Widgets

Plug-and-play embeddable widgets for any website. Zero dependencies. Just one script tag.

This is a self-contained demo file. Download and open it anywhere — it works offline.

1. Donation Widget

A universal support widget with preset amounts ($5, $10, $25, $50), custom amount input, one-time/monthly toggle, trust signals, and a CTA that links to your support page.

Sticky PlacementFixed bottom-left green pill that expands into the full donation card on click.
Inline PlacementRenders the donation card directly in page flow, wherever the script tag is placed.

Live Demo — Inline Widget

Interactive widget below — try it

Embed Code — Inline

<!-- Wordsmyth Donation Widget (inline) --> <script src="https://kids.wordsmyth.net/widget/donate.js" data-support-url="https://kids.wordsmyth.net/support" data-placement="inline"></script>

Embed Code — Sticky

<!-- Wordsmyth Donation Widget (sticky pill) --> <script src="https://kids.wordsmyth.net/widget/donate.js" data-support-url="https://kids.wordsmyth.net/support" data-placement="sticky" data-delay="3000"></script>

Configuration

AttributeDefaultDescription
data-support-url/supportWhere "Continue" button links to
data-placementstickysticky or inline
data-delay3000Delay (ms) before sticky pill appears
data-target—CSS selector for inline container
data-utmdonation-widget-embedUTM source for analytics

2. Go Ads-Free Widget

Automatically finds every ad on the page and overlays a witty "Go Ads-Free" bar. Detects Google Ads, Amazon, Mediavine, Freestar, and 40+ ad network patterns.

Auto-DetectionScans for ads by class, ID, data attributes, and iframe sources. Zero config needed.
Responsive OverlaysWide ads get witty text + green CTA. Narrow ads get a compact CTA-only bar.

Embed Code

<!-- Wordsmyth Go Ads-Free Widget --> <script src="https://kids.wordsmyth.net/widget/go-ad-free.js" data-subscribe-url="https://kids.wordsmyth.net/subscribe"></script>

Configuration

AttributeDefaultDescription
data-subscribe-url/subscribeWhere CTA links to
data-delay1500Delay (ms) before first scan
data-poll3000Re-scan interval (ms). 0 to disable.
data-selectors—Extra CSS selectors (comma-separated)

3. Use Both Together

Drop both script tags on any page. The donation widget handles contributions, the Go Ads-Free widget converts ad impressions into subscribers.

Combined Embed Code

<!-- Wordsmyth Donation Widget (sticky) --> <script src="https://kids.wordsmyth.net/widget/donate.js" data-support-url="https://kids.wordsmyth.net/support" data-placement="sticky"></script> <!-- Wordsmyth Go Ads-Free Widget --> <script src="https://kids.wordsmyth.net/widget/go-ad-free.js" data-subscribe-url="https://kids.wordsmyth.net/subscribe"></script>
Zero DependenciesPure vanilla JS. No React, jQuery, or framework. Works on any website.
Lightweight~8KB each. Self-contained CSS. No external requests.
Analytics ReadyFires Google Analytics (gtag) events for impressions, clicks, and dismissals.
ConfigurableAll options via data-* attributes. No code changes needed.
